Reduced-order unsteady transonic analysis for large aspect-ratio wings

 

Analysis of unsteady transonic flow is a computationally challenging process and usually approached using latest computational fluid dynamics (CFD) tools. Application of such procedure for coupled fluid-structure interaction, especially for aircraft design and optimization is thus not possible. A new approach is being developed by using indicial functions which can predict accurate unsteady airfoil behavior for attached flows. For large aspect-ratio wings the present approach can be applied via strip theory thereby reducing several order of states from the full-order three dimensional CFD model.

 

Unsteady lift obtained form the indicial functions due to harmonic pitching show good correlation of with Reynold Averaged Navier-Stokes (RANS) simulation performed via SU2 for a 8% thick BACJ airfoil.
